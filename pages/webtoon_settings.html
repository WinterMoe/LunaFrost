Google Cloud Vision API

Option 1: Simple API Key (Recommended - Easier):
1. Go to Google Cloud Console
2. Select your project (or create one)
3. Enable "Cloud Vision API" (APIs & Services ‚Üí Enable APIs ‚Üí Search "Cloud Vision API")
4. Go to "APIs & Services" ‚Üí "Credentials"
5. Click "Create Credentials" ‚Üí "API Key"
6. Copy the API key and paste it above
7. (Optional) Click "Restrict Key" ‚Üí Restrict to "Cloud Vision API" for security

Option 2: Service Account JSON (For advanced users):
Upload a service account JSON file to your server and enter the full path (e.g., /var/www/translator/google-credentials.json)

Free tier: 1,000 requests/month

Azure Computer Vision API

How to get it:
1. Go to Azure Portal
2. Click "Create a resource" ‚Üí Search for "Computer Vision" ‚Üí Click "Create"
3. Fill in the form: - Name: Choose a name (e.g., "my-vision-api") - Subscription: Select your subscription - Location: Choose a region - Pricing tier: Select "Free F0" (5,000 requests/month)
4. Click "Review + create" ‚Üí "Create"
5. Wait for deployment, then click "Go to resource"
6. Find your credentials (try these locations):

  • Look for "Keys and Endpoint" in the left sidebar menu
  • Or look for "Quick start" ‚Üí Keys and endpoint are shown there
  • Or look for "Resource Management" ‚Üí "Keys and Endpoint"
  • Or in the Overview page, look for a section called "Show access keys" or "Manage keys"
  • Sometimes it's under "Settings" ‚Üí "Keys"
7. You'll see: - KEY 1 or KEY 2 (copy either one) ‚Üí Paste in "API Key" field above
- Endpoint (copy the URL, looks like https://your-resource.cognitiveservices.azure.com/) ‚Üí Paste in "Endpoint" field above
Free tier: 5,000 requests/month
Note: If you still can't find it, try clicking on your resource name in the top breadcrumb, then look in the left sidebar for "Keys and Endpoint"
